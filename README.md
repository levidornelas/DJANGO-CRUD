# Cadastrando Usuários:

Realizando o CRUD (Create, Read, Update, Delete) em aplicações WEB, usando do framework Django para a realização do backend. Além das implementações básicas já citadas, há a checagem de login do usuário para que as ações possam ser realizadas. Caso ele não esteja logado por meio do '/admin', o CRUD não é possível.


## Rodando localmente

Clone o projeto

```
  git clone https://github.com/levidornelas/DJANGO-CRUD
```

Instale as dependências:
```
  pip install -r requirements.txt
```

Inicie o servidor

```
  python manage.py runserver
```
